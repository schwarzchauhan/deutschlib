# deutschlib

### How to run project
```bash
npm run dev
# or 
npm start
```


### About appliation
- a user can register in to app
- then she can login 
- user can also add their skill in profile section
- a user can have multiple skills ([populate](https://mongoosejs.com/docs/populate.html#populate) concept used here)
